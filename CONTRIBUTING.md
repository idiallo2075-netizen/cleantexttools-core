# Contributing to CleanTextTools Core

Thank you for your interest in contributing to CleanTextTools Core.

We welcome improvements that help make the project more reliable, accessible, and easier to use.

## Ways to Contribute

You can contribute by:

- Fixing bugs
- Improving documentation
- Enhancing accessibility
- Improving performance
- Writing tests
- Suggesting new cleaning rules
- Improving the user interface

## Before You Start

Please keep contributions focused on the **Core edition**.

This repository is intentionally limited and does **not** include:

- Proprietary production code
- Private infrastructure
- Internal AI prompts
- SEO strategy
- Analytics configuration
- Commercial features

## Pull Requests

Before submitting a pull request:

1. Keep changes focused on a single topic.
2. Test your changes.
3. Update documentation if necessary.
4. Write clear commit messages.

## Code Style

Please:

- Write clean, readable code.
- Use meaningful variable names.
- Keep functions small and maintainable.
- Add comments only where they improve understanding.

## Feature Requests

Feature requests are welcome.

Please open a GitHub Issue describing:

- The problem
- Your proposed solution
- Why it would benefit the project

## Code of Conduct

Please be respectful and constructive when participating in discussions.

We aim to maintain a welcoming environment for everyone.

Thank you for helping improve CleanTextTools Core.
