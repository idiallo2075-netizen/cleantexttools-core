# Security Policy

## Supported Versions

The latest version of CleanTextTools Core is supported for security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** report it through a public GitHub issue.

Instead, report it through the official CleanTextTools support page:

https://cleantexttools.com/contact/

Please include:

- A clear description of the issue
- Steps to reproduce the problem
- The affected file or feature
- The potential security impact
- Any suggested remediation (if available)

Do **not** include:

- API keys
- Passwords
- Tokens
- Personal information
- Customer data
- Proprietary source code

## Scope

This repository contains only the public open-source Core edition of CleanTextTools.

Security issues related to the hosted website or proprietary services should also be reported using the official contact page.

Thank you for helping improve the security of CleanTextTools.
